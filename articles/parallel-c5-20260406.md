# TSL2561 光照度自動曝光控制：0.1–40k Lux 三段積分時間實作

## 問題場景

你在室內拍商品照，窗外陽光從窗簾縫隙滲入。相機測光錶顯示 EV 12，但 Arduino 環境光監控只能讀到「明亮」兩個字。用 LDR 測光，指數容易飄移，Lux 值根本算不出來。

TSL2561 是數位光感測器，支援 **0.1–40,000 Lux** 動態範圍，輸出為標準 I2C 訊號，解決 LDR 的非線性問題。

## 現有方案問題

LDR 有三個核心限制：
- 非線性響應：Lux 與電阻值呈 log 關係，需 lookup table 校正
- 10-bit ADC 解析度不足：最小刻度約 4.9mV
- 無法更換積分時間：暗光場景容易饱和

TSL2561 的核心優勢在於**三段積分時間**（13.7 / 101 / 402 ms）。積分時間越長，弱光下解析度越高，但强光時容易饱和。這個特性讓 TSL2561 可當作 DIY 測光表使用。

## 核心原理

TSL2561 內建兩個光二極體：CH0（可見+紅外）和 CH1（僅紅外）。積分時間決定 ADC 的最大 count 值：
- 13.7 ms：max count ≈ **60,000**（強光）
- 402 ms：max count ≈ **40,000**（弱光）

饱和偵測門檻設在 **60,000 counts**。超過代表曝光過度，須縮短積分時間。

自動曝光邏輯：當 CH0 < 1000（弱光門檻）且目前為 13.7ms，切換至 402ms；當 CH0 > 60,000 且目前為 402ms，切換至 13.7ms。

## 材料清單

- Arduino UNO × 1：主控制器
- TSL2561（CN-41 封裝）× 1：數位光感測器
- 4.7kΩ 上拉電阻 × 2：I2C SCL / SDA
- 跳線 × 4

## 製作步驟

步驟1：接線
- VCC → 3.3V（TSL2561 支援 3.3V）
- GND → GND
- SCL → A5
- SDA → A4
- 4.7kΩ 上拉電阻接 SCL / SDA 到 3.3V

步驟2：安裝 Adafruit TSL2561 函式庫
- Arduino IDE → 草稿簿 → 包含函式庫 → 管理函式庫 → 搜尋「Adafruit TSL2561」並安裝

步驟3：上傳以下程式碼

```arduino
// 檔案說明：TSL2561 自動曝光控制，積分時間動態切換
// 作者：HF Chang
// 硬體：Arduino UNO + TSL2561（CN-41）

#include <Wire.h>
#include <Adafruit_TSL2561.h>

Adafruit_TSL2561 tsl = Adafruit_TSL2561(TSL2561_ADDR_FLOAT, 12345);

void setup() {
  Serial.begin(9600);
  if (!tsl.begin()) {
    Serial.println("TSL2561 未找到");
    while (1);
  }
  tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_13_7MS);
  Serial.println("初始積分時間：13.7ms");
}

void loop() {
  uint16_t ch0 = tsl.getLuminosity(TSL2561_FULLSPECTRUM);
  uint16_t ir = tsl.getLuminosity(TSL2561_INFRARED);
  float lux = tsl.calculateLux(ch0, ir);

  tsl2561_autoExposure(ch0);

  Serial.print("CH0: "); Serial.print(ch0);
  Serial.print(" | Lux: "); Serial.println(lux, 2);
  delay(500);
}

void tsl2561_autoExposure(uint16_t ch0) {
  tsl2561_integration_t integration = tsl.getIntegrationTime();

  if (ch0 > 60000 && integration != TSL2561_INTEGRATIONTIME_13_7MS) {
    tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_13_7MS);
    Serial.println("強光模式：積分時間 13.7ms");
  } else if (ch0 < 1000 && integration != TSL2561_INTEGRATIONTIME_402MS) {
    tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_402MS);
    Serial.println("弱光模式：積分時間 402ms");
  }
}
```

## 測試驗證

打開 Serial Monitor，觀察 CH0 與 Lux 值：
- 室內一般照明：CH0 約 3000–5000，Lux 約 **300–500**
- 强光照射：CH0 超過 60000，自動切換至 13.7ms，Lux 穩定
- 黑暗中：CH0 < 1000，自動切換至 402ms，微弱光線也能測出

## 常見問題

問題：Serial Monitor 顯示 "TSL2561 未找到"
原因：I2C 接線鬆脫或位址錯誤（浮動位址預設 0x39）
解法：用 I2C Scanner 確認 0x39 存在，檢查 SCL / SDA 線

問題：Lux 計算值為 0
原因：積分時間與增益設定不匹配
解法：確認初始設定為 `setIntegrationTime(TSL2561_INTEGRATIONTIME_13_7MS)`
