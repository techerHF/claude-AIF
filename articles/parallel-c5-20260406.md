# TSL2561 自動曝光控制：0.1–40k lux 實作

## 問題場景

園藝自動澆水系統上線後，光感測模組在陰天與直射陽光下輸出值差異過大。同一顆光敏電阻（LDR），陰天輸出 200、陽天輸出 800。

LDR 的電阻值隨照度呈對數關係，必須做 ADC 類比讀取。室內場景常見 0–500 lux，但窗邊直射陽光可達 **20,000 lux** 以上。LDR 在高照度區域解析度不足，是自動曝光失敗的主因。

## LDR 限制與 TSL2561 差異化

LDR 有三個核心限制：

1. **非線性響應**：lux 與電阻值呈 log 關係，需查表或擬合曲線才能轉換
2. **類比輸出**：10-bit ADC 在 0–5V 範圍內最小刻度約 4.9mV
3. **無整合時間控制**：無法延長感測時間以捕捉微弱光訊號

TSL2561 是一款 I2C 數位光感測器，封裝為 CN-41。

關鍵規格：
- 光譜響應：可見光（400–700nm），接近人眼響應
- 動態範圍：**0.1–40,000 lux**，室內到陽光直射全涵蓋
- 積分時間：**13.7ms / 101ms / 402ms** 三段可調
- 輸出格式：32-bit 數位（紅外 + 全光譜通道）

數位輸出直接是 count 值，不必做類比轉換。積分時間越長，對弱光的靈敏度越高。

## 積分時間與 Lux 計算原理

TSL2561 內建兩個光二極體：CH0（全光譜）與 CH1（紅外）。

 lux 計算公式（Adafruit TSL2561 library 預設演算法）：

```
lux = 0.3 * (CH0 - CH1) / CH0
```

積分時間影響 count 值的量級：相同的照度下，402ms 積分得到的 CH0 值約是 13.7ms 的 **29 倍**。因此，程式中必須根據積分時間設定對應的 GAIN（增益），否則 lux 計算會差到 29 倍。

自動曝光控制邏輯：
- 積分時間 13.7ms：適用於室外強光（> 10,000 lux）
- 積分時間 402ms：適用於室內弱光（< 100 lux）
- 軟體偵測 CH0 是否饱和（count = 65535），自動切換積分時間

## 材料清單

- Arduino Uno × 1：主控制器
- TSL2561（TSSOP-5 封裝）× 1：數位光感測器
- 4.7kΩ 上拉電阻 × 2：I2C SDA/SCL（部分模組已內建）
- 麵包板 × 1：快速驗證用

## 製作步驟

步驟1：硬體接線
TSL2561 VCC 接 3.3V，GND 接 GND，SDA 接 A4，SCL 接 A5。I2C 需要共同接地（3.3V 與 5V 系統共地）。

步驟2：安裝函式庫
Arduino IDE → 草稿碼 → 引用函式庫 → 管理函式庫 → 搜尋「Adafruit TSL2561」→ 安裝。

步驟3：上傳測試程式
預期輸出：
```
Auto Gain: 1x, Integration Time: 402ms
Lux: 127.34
Lux: 254.67
```
室內正常照度（300–500 lux）約落在 100–300 範圍。

步驟4：驗證自動曝光
用手遮住感測器，TSL2561 應自動切換至長積分時間，lux 值應緩慢上升而非驟降。

## 程式碼

```arduino
// TSL2561 自動曝光控制
// 硬體：Arduino Uno + TSL2561（I2C，位址 0x39）

#include <Wire.h>
#include <Adafruit_TSL2561.h>

Adafruit_TSL2561 tsl = Adafruit_TSL2561(TSL2561_ADDR_FLOAT, 12345);

void setup() {
  Serial.begin(9600);
  if (!tsl.begin()) {
    Serial.println("TSL2561 初始化失敗");
    while (1);
  }
  tsl.enableAutoRange(true);
  tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_402MS);
  Serial.println("TSL2561 自動曝光模式啟動");
}

void loop() {
  float lux = tsl.getLuminosity(TSL2561_VISIBLE);
  Serial.print("Lux: ");
  Serial.println(lux, 2);

  // 饱和偵測：手動切換積分時間
  uint32_t ch0 = tsl.getLuminosity(0);
  if (ch0 >= 60000 && tsl.getIntegrationTime() != TSL2561_INTEGRATIONTIME_13_7MS) {
    tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_13_7MS);
    Serial.println("強光模式：積分時間 13.7ms");
  } else if (ch0 < 1000 && tsl.getIntegrationTime() != TSL2561_INTEGRATIONTIME_402MS) {
    tsl.setIntegrationTime(TSL2561_INTEGRATIONTIME_402MS);
    Serial.println("弱光模式：積分時間 402ms");
  }
  delay(500);
}
```

上傳方式：Arduino IDE 1.8+ → 板型選 Arduino Uno → 速率 9600 baud。

## 常見失敗原因

問題：Lux 值一直是 0
原因：TSL2561 未正確通電，或 I2C 位址錯誤（浮動位址預設為 0x39）
解法：用 I2C Scanner 確認 0x39 是否存在

問題：室外 Lux 值遠低於預期
原因：積分時間仍設為 402ms，在強光下饱和後輸出錯誤值
解法：確認 saturation 偵測邏輯正常觸發，切換至 13.7ms
