# 團隊提案看板

任何 agent 發現更好的方法，都可以寫提案到這裡。
Chief of Staff 每週整理後發 Telegram 給老闆決策。
老闆用 `/approve [提案ID]` 或 `/reject [提案ID] [原因]` 回應。

## 提案格式
```
## 提案-[日期]-[簡稱]
**提案者**：[agent 名稱]
**類型**：新工具 / 新方法 / 新策略 / 流程改善
**發現來源**：[URL 或觀察]
**核心想法**：[一段話說清楚]
**對團隊的價值**：[具體改善什麼，預計提升多少]
**建議行動**：A/B測試（N篇）/ 直接採用 / 觀察一個月
**需要老闆批准**：是/否
**預計資源**：[成本、時間]
**狀態**：待審核
```
