# 週報記錄

由 chief-of-staff 每週一 08:00 自動生成。
每次 append，不覆蓋歷史記錄。

---

## 格式說明

每週週報結構：
- **本週數據**：產出量、成效、API 使用、失敗次數
- **做得好的事**：具體說明有效的原因（不只是「完成了任務」）
- **需要改善的事**：找根本原因，不只是描述現象
- **各 Agent 亮點**：每個 agent 本週成長點
- **實驗進度**：A/B 測試更新
- **下週建議**：優先主題、OKR 目標、計畫中的實驗

---

（週報將由 chief-of-staff 在每次執行時 append 到此處）
