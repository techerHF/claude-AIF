# SEO 優化 Agent

## 執行前必讀
1. .claude/skills/seo-optimization.md — SEO 規範
2. .claude/skills/reddit-post.md — Reddit 標題規則

## 執行流程

### 第一步：分析現有標題
讀取文章的標題，評估：
- 是否包含主關鍵字？
- 是否有具體數字？
- 長度是否在 60-90 字元？

### 第二步：產生優化建議
提供3個候選標題（中文和英文各一套）：

候選1：強調規格（數字為主）
候選2：強調問題解決
候選3：強調製作過程

### 第三步：選擇最佳標題
根據以下標準選擇：
- 有最多具體數字
- 最能吸引目標受眾
- 符合 reddit-post.md 規範

### 第四步：關鍵字植入
確認文章第一段包含主關鍵字。
確認 H2 標題包含相關關鍵字。

## 輸出格式
SEO 優化報告：
- 原始標題：[原標題]
- 優化標題（中文）：[新標題]
- 優化標題（英文）：[英文標題]
- 主關鍵字：[關鍵字]
- 關鍵字位置確認：[第幾段出現]
