# 折扣與定價策略

## 定價結構
第1層 $9：單篇深度指南
  → 一個感測器的完整設計思考
  → 進入門檻低，用來建立信任

第2層 $19：完整專案包
  → 多感測器組合 + 狀態機設計
  → 給已有基礎、想更完整的人

第3層 $5/月：訂閱制
  → 每月新主題，培養跨領域設計思維
  → 訂閱者優先試用新產品

## 定價心理
$9 的意義：「比一杯咖啡貴不了多少」
→ 不需要猶豫的價格點，降低第一次購買阻力

$5/月 的意義：「每週不到 $2」
→ 讓人覺得划算，容易續訂

$19 的意義：「完整方案，有具體成果」
→ 對比 $9，升級感明確

## 折扣時機（不能隨便打折）
只有以下情況才折扣：
1. 新產品上架前72小時（Early Access $7）
2. 訂閱者專屬（現有訂閱者購買第1層或第2層享85折）
3. 季節性（每年一次，黑色星期五最多7折）

## 折扣文案原則
不說「限時特價！」
說「訂閱者專屬，72小時內有效」
不用倒數計時營造恐慌
用「這個月新主題是 XX，訂閱者可以先看」

## 如何在文章中自然帶入定價
第七層進階應用結尾：
「如果你想把這個感測器跟距離感測器組合成多模態判斷系統——
這進入了互動設計的核心邏輯。
我把完整的設計框架整理成指南，包括三種情境的狀態機設計。
[Whop 連結]，$9。」

禁止：
「只要 $9 你就能學到...」（強推銷）
「不買後悔！」（恐嚇）
「超值！」（空洞）

## Whop vs Reddit 規則
Reddit 主文：絕對不放 Whop 連結
Reddit 留言第一則：可以放，格式固定
Medium：在第七層段落放，一篇最多一個連結
Amazon 聯盟：在材料清單放，每篇最多5個

## 產品推出順序對應的 Skill 使用
階段1（Reddit 跑通）：
  只用 writing-style, article-structure, reddit-post
  不用任何 Whop skill

階段2（上架第1層）：
  加入 whop-product, monetization-strategy
  確認有 Reddit 留言互動再上架

階段3（上訂閱）：
  加入 discount-pricing, whop-copy
  現有讀者是第一批告知對象

階段4（上第2層）：
  訂閱者是第一批買家
  用 discount-pricing 設定訂閱者專屬折扣
