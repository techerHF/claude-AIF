# Reddit 留言互動策略

## 留言目的
建立可信度，避免被認定為純廣告帳號。

## 發文前預熱（每個 subreddit）
在目標 subreddit 發文前，先留3-5則有價值的留言：
- 回答別人的技術問題
- 分享相關資源（不是自己的）
- 給初學者實用建議

## 發文後留言策略

### 第一則留言（立即發）
固定格式：
「Full writeup with code, circuit diagram, and files:
[Payhip 連結]

Happy to answer questions about the build process.」

### 第二則留言（等有人留言後）
回應第一個問題，詳細說明技術細節。
展示你真的懂這個主題。

### 留言互動原則
- 每個問題都回應（前48小時）
- 技術問題給具體答案（數值、公式）
- 不催促購買
- 感謝讚美時簡短回應就好

## 防封號規則
- 不在同一天的多篇文章裡放連結
- 留言不能比主文短太多（太短顯得像機器人）
- 不複製貼上相同留言到不同貼文
- 帳號要有正常的瀏覽和投票歷史

## 自動化留言的限制
第一則留言可以用模板。
第二則之後必須根據實際問題回應（需要人工或另外的 agent 處理）。
