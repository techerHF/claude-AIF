# SEO 優化規範

## 標題優化

### 關鍵字策略
主關鍵字（每篇選1個）：
- Arduino pressure sensor
- capacitive sensor DIY
- tactile feedback Arduino
- ESP32 sensor project
- PCB design beginner

輔助關鍵字（每篇選2-3個）：
- homemade, DIY, low cost, tutorial
- step by step, complete guide
- copper foil, PDMS, flexible sensor

### 標題結構
[主關鍵字] + [具體規格/數字] + [結果]

範例：
「Arduino Pressure Sensor with Tactile Feedback - 0 to 20 kPa, 2-Level Detection」

## 文章內 SEO

### 關鍵字密度
- 主關鍵字出現3-5次
- 第一段必須出現主關鍵字
- 不要硬塞，要自然融入句子

### 段落結構 SEO
- 使用 H2、H3 標題
- 每個標題包含關鍵字或相關詞
- 清單格式（bullet points）比長段落更容易被索引

### 數字的 SEO 價值
標題和段落中的具體數字提高點擊率：
- 「減少33%的攻擊成功率」比「減少很多」好
- 「5個步驟」比「幾個步驟」好
- 「成本200元以內」比「低成本」好

## Reddit 特定 SEO
- 標題出現感測器型號（如 ADS1115、SI5351）
- 提到具體材料（如 PDMS、copper foil）
- 包含數字（kPa、Hz、pF）

## 禁止的 SEO 行為
- 關鍵字堆砌（同一詞連續出現超過3次）
- 標題與內容不符
- 用點擊誘餌（clickbait）標題
