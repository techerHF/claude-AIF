# 圖片與電路圖說明規範

## 電路圖說明格式
每張電路圖旁邊必須有文字說明：

格式：
「圖N：[電路名稱]
接線說明：
- [元件A] 的 [接腳] 連接到 [元件B] 的 [接腳]
- [元件C] 的 VCC 接 5V，GND 接地
注意：[最重要的注意事項]」

## Alt Text 格式（Reddit 上傳圖片時）
格式：「[電路類型] showing [關鍵元件] connections for [功能描述]」
範例：「Capacitive pressure sensor circuit showing ADS1115 ADC connections for dual-level pressure detection」

## 接線圖必須包含
- 所有元件都有標籤（型號）
- 電源線（VCC/GND）要清楚標示顏色
- 信號線和電源線用不同顏色
- 有比例尺或尺寸參考

## 製作過程照片說明
每張照片說明格式：
「步驟N：[動作]
照片顯示：[照片中看到什麼]
重點：[這個步驟最重要的細節]」

## Openverse 圖片使用規範
- 只使用 CC0 或 CC-BY 授權圖片
- 必須附上作者和來源：「圖片來源：[作者名稱]，[授權類型]，Openverse」
- 圖片必須與內容直接相關

## 禁止
- 沒有說明的電路圖
- 看不清楚接線的照片
- 缺少型號標示的元件
- 使用未授權圖片
